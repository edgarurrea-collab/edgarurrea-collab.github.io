2. Comandos Principales de la Terminal

Para la clonación de la solución, la compilación de binarios mediante herramientas de automatización y el posterior inicio del sistema operativo, ejecute la siguiente secuencia:

Clonación del Repositorio de Trabajo:

git clone https://github.com/juannunezbenitez-create/laboratorio-so-grupo7.git
cd laboratorio-so-grupo7

Compilación del Software:

make all

Ejecución del Programa Principal:

./main_executable --config=default.conf

Conclusiones

A partir del desarrollo práctico del laboratorio, se han deducido los siguientes puntos concluyentes en relación a la arquitectura y metodología de trabajo implementada:

Área de Análisis Conclusión del Proyecto
Control de Versiones La distribución organizada de ramas y el flujo continuo de commits permitieron mitigar conflictos de código y asegurar la trazabilidad del software.
Sincronización y Procesos El uso óptimo de primitivas del sistema operativo redujo el sobrecosto de CPU y garantizó la integridad de los datos compartidos.
Documentación Estática El despliegue del sitio en plataformas de hosting rápido unifica los criterios de evaluación y facilita la auditoría del proyecto por terceros.
